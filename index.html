Getting Started

JVEscriptext is a scripting language designed to work with HTML, CSS, and JavaScript in a browser environment. You can use it to control the DOM, run JS functions, and manage game logic or interactive web apps.

Variables

var player = "Joseph"
var score = 10

Use variables to store values. Strings should be wrapped in quotes.

Functions

function greet() {
  print "Hello " + player
}

greet()

Functions let you reuse code and run multiple commands together.

Events

event start {
  print "Game started!"
}

event click {
  print "Button clicked!"
}

Events allow JVE to respond to triggers like page load or user actions.

HTML Commands

html.text "title" "Hello Player"
html.style "title" "color:red"
html.create "button" "myBtn"
html.append "myBtn" "body"

Control HTML elements: change text, style, or create new elements.

JavaScript Commands

js.call "sayHello"

Call JavaScript functions directly from JVE scripts.
