Getting Started

JVEscriptext lets you control HTML, CSS, and JS via JVE scripts. Great for games, interactive apps, or web logic.

Variables

var player = "Tester"
var score = 10

Functions

function greet() {
print "Hello " + player
}
greet()

Events

event start {
print "Game started!"
}
event click {
print "Button clicked!"
}

HTML Commands

html.text "title" "Hello Player"
html.style "title" "color:red"
html.create "button" "myBtn"
html.append "myBtn" "body"

JavaScript Commands

js.call "sayHello"
